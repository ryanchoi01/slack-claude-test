# Slack Claude Bot

Slack에서 Claude AI를 통해 코드 작업을 요청할 수 있는 봇입니다.

## 개요

Slack 채널에 메시지를 보내면 Claude가 코드를 분석하고, 수정하고, PR을 생성하는 등의 작업을 자동으로 수행합니다.

## 주요 기능

### 코드 수정 및 작성
- Slack 메시지로 코드 변경을 요청하면 Claude가 직접 파일을 수정합니다.
- 새 기능 추가, 버그 수정, 리팩토링 등 다양한 작업을 처리합니다.

### 브랜치 및 PR 자동 생성
- 작업마다 새로운 브랜치를 자동으로 생성합니다.
- 브랜치 명명 규칙: `claude-bot/{날짜}-{작업내용}-{해시}`
- 변경 사항을 커밋하고 GitHub에 PR을 생성합니다.

### 파일 탐색 및 분석
- 코드베이스를 탐색하여 관련 파일을 찾고 분석합니다.
- 파일 내용 읽기, 패턴 검색, 구조 파악 등을 수행합니다.

### 코드베이스 이해
- 기존 코드 패턴과 아키텍처를 파악하여 일관성 있는 코드를 작성합니다.
- 프로젝트 컨벤션에 맞춰 변경 사항을 적용합니다.

## 사용 방법

Slack 채널에서 봇을 멘션하거나 지정된 채널에 메시지를 보내세요.

**예시 요청:**
```
app.py에 farewell 함수 추가해줘
README 업데이트해줘
greet 함수에 에러 처리 추가해줘
```

## 프로젝트 구조

```
slack-claude-test/
├── app.py        # 샘플 애플리케이션
└── README.md     # 프로젝트 문서
```
